Для чего хорош С++ ?

  1. Можно работать с памятью на низком уровне
  2. Нет «виртуальных машин» или фреймворков, которые занимаются, например, сборкой мусора или выделением памяти
  3. Быстрое (иногда очень быстрое) выполнение кода по сравнению с более высокоуровневыми языками (Python, C#, Java и другими)
  4. Полный доступ к API операционной системы без обёрток (в которых может быть реализовано не всё)
